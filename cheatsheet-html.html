Cheatsheet - HTML

Guía para escribir HTML5 moderno de un modo semántico: estructura, texto, enlaces, formularios, listas, etc ... de un modo accesible y que sigan buenas prácticas de maquetación y diseño.

  • Estructura
  • Texto
  • Enlaces
  • Formularios
  • Listas

Estructura

El código HTML se organiza en elementos, que se representan mediante etiquetas. La estructura básica de un documento HTML5 es la siguiente:

                  <!DOCTYPE html>
                      <html lang="es">
                        <head>
                          <meta charset="UTF-8">
                          <meta name="viewport" content="width=device-width, initial-scale=1.0">
                          <title>Título de la página</title>
                        </head>
                        <body>
                          Contenido de la página
                        </body>
                      </html>

En esta estructura, el elemento raíz es <html> </html>, que contiene dos secciones principales: <head> y <body>. El elemento <head> incluye metadatos, enlaces a hojas de estilo, scripts, etc., mientras que el elemento <body> contiene el contenido visible de la página.
